# 店舗マスタ参照

review_observation（mart_gbp）で使う **store_code** は、既存の店舗マスタ **BigQuery `ikeuchi-ga4.stg_freee_prd.dim_store`** の **store_id** と一致させる。

---

## dim_store のスキーマ（主要列）

| 列名 | 型 | 説明 |
|------|-----|------|
| store_id | INTEGER | 店舗ID（**store_code はこの値を文字列にしたもの**） |
| store_name | STRING | 店舗名（例: 川越, 平塚, 越谷, 江戸川） |
| section_id | INTEGER | 部門ID |
| shortcut | STRING | 短縮名（NULL のことが多い） |
| … | … | その他（memo, site_area_sqm, headcount_* 等） |

- **store_code**（mart_gbp の places_provider_map / performance_* で使用）  
  = **CAST(dim_store.store_id AS STRING)**  
  例: 川越 → `'3547880'`

---

## 川越・平塚・越谷・江戸川の対応（確認済み）

| store_name（dim_store） | store_id | store_code（mart_gbp で使用） |
|-------------------------|----------|-------------------------------|
| 川越 | 3547880 | 3547880 |
| 平塚 | 3547881 | 3547881 |
| 越谷 | 3547882 | 3547882 |
| 江戸川 | 3547883 | 3547883 |

※ 同じ名前の別店舗（例: 藤沢・平塚店 3734437、越谷・見沼店 3734439、江戸川松江 3568911）は別 store_id。既存月次データの「川越・平塚・越谷・江戸川」は上記4件に対応する想定。

---

## places_provider_map と provider_place_id（GBP 上の識別子）

- **provider_place_id** は **GBP 上の識別子**（Google Business Profile の location リソース名）を格納する列である。  
  論理名: **GBP location リソース名**。形式例: `accounts/{accountId}/locations/{locationId}`。
- 現在、`sql/040_insert_places_provider_map.sql` で一括登録した行は、**GBP の実データ（location 一覧）を見ていない状態**で投入したため、**provider_place_id は空文字 `''` のまま**である。正しい GBP 識別子は未設定。
- GBP の location 一覧（各店舗の `accounts/.../locations/...`）が分かり次第、**store_code と対応付けて provider_place_id を UPDATE** する必要がある。  
  例: `UPDATE mart_gbp.places_provider_map SET provider_place_id = 'accounts/123/locations/456' WHERE store_code = '3547880' AND provider = 'google';`  
  **取得手順**: [GBPの店舗コード取得方法.md](GBPの店舗コード取得方法.md) を参照。

---

**既存月次インポート**: dim_store に存在する行は**全て**インポート対象とする。全 store_id について 2023-07/08/09 の行を挿入し、川越・平塚・越谷・江戸川は実データ、それ以外は metrics を NULL で投入する。SQL は `scripts/gen_020_import_monthly.py` で生成（[既存月次データのインポート.md](既存月次データのインポート.md) 参照）。

## 参照クエリ例

```sql
-- 店舗名で store_code を確認したい場合
SELECT store_id, store_name,
       CAST(store_id AS STRING) AS store_code
FROM `ikeuchi-ga4.stg_freee_prd.dim_store`
WHERE store_name IN ('川越', '平塚', '越谷', '江戸川')
ORDER BY store_name;
```
